Curso de HTML5 E CSS3

A HTML É uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir o conteudo da página web,e suas especificações são controladas pela W3C (World Wide Web Consortium). Com as CSS criamos as configurações visuais que um sute vai ter, o CSS3 é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML, ou seja, definir a aparência das páginas, para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais.
